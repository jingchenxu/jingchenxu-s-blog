## vim

> 生命不息 折腾不止

Esc:进入命令模式

：+ wq --&gt; 退出并保存修改

：+ q! --&gt; 退出不保存修改

r --&gt; 修改光标所在位置内容

i --&gt; 在光标所在位置插入内容

vim file1 file2 --&gt; vim同时打开多个文件

：open file --&gt; 在vim页面打开文件

：split --&gt; 纵向显示多个文件

：vsplit --&gt; 横向显示多个文件

：bn --&gt; 下一个文件

：bp --&gt; 上一个文件

Ctrl+方向键 --&gt; 文字行的起始点

命令模式下输入 / + "asdfasdf" 则表示全文搜索
