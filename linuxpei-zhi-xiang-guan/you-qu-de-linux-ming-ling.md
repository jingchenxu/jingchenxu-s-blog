# 有趣的Linux命令

* tree

这是一个我看到很多的命令，在很多的文档中都能看到此命令的生成结果，而且在win10下此命令是原生支持的，在Linux下可能需要安装一下，安装的命令为：`shell sudo apt-get install tree` 然后他的效果是这样的：
